## 🛠️ The Core Tech Stack Zone

Every app in this challenge is built on the exact same architecture — the constraint is the point.

| Layer                         | Technology                                   |
| ----------------------------- | -------------------------------------------- |
| 📱 **Mobile & Web Frontends** | React Native (Expo) & Next.js                |
| ⚙️ **Backend & Deployment**   | Next.js API Routes → GCP Cloud Run / Vercel  |
| 🗄️ **Database**               | Neon.tech (Serverless Postgres + Prisma ORM) |
| 🔐 **Authentication**         | Neon Auth (Unified Auth)                     |
| 📦 **File Storage**           | Cloudflare R2 (Zero-Egress S3 Storage)       |
| 🧱 **Monorepo Setup**         | Turborepo per project                        |
